🌾 Argrithm API Docs

Hyperlocal agricultural news, Gemini Flash AI advisory, speech-to-text, and text-to-speech — all in one system built for Indian farmers.

Base URL http://127.0.0.1:8000
Quick Start: Start the server with uvicorn server:app --reload --port 8000, then use the clickable examples below or open Swagger UI.
📰 News Scraper API
GET

/news

Fetch hyperlocal, categorized agricultural news
Query Parameters
ParamTypeRequiredDescription
citystringRequiredFarmer's city or village (e.g. Ongole)
cropstringRequiredCrop the farmer grows (e.g. rice)
statestringOptionalFarmer's state — improves regional accuracy
languagestringOptionalen / te / hi / mr / ta / kn / ml
limitintegerOptionalMax articles per category (1–15, default 5)
Clickable Examples
🌾 Rice farmer in Ongole, Telugu news
GET /news?city=Ongole&crop=rice&state=Andhra+Pradesh&language=te&limit=5
▶ Run in Browser
🌾 Wheat farmer in Ludhiana, Hindi news
GET /news?city=Ludhiana&crop=wheat&state=Punjab&language=hi&limit=5
▶ Run in Browser
🍇 Grapes farmer in Nashik, Marathi news
GET /news?city=Nashik&crop=grapes&state=Maharashtra&language=mr
▶ Run in Browser
Response Schema
200 OK
"query": { "city": "Ongole", "crop": "rice", "state": "Andhra Pradesh" }, "total_articles": 18, "weather": { "temperature": 27.4, "humidity": 80, "condition": "Few clouds" }, "llm_summary": "Current weather in Ongole: Few clouds, 27°C...", "crop_news": [ { "title": "...", "url": "...", "score": 8 } ], "weather_news": [ ... ], "market_news": [ ... ], "pest_alerts": [ ... ], "govt_schemes": [ ... ]
🤖 Gemini Flash AI
POST

/gemini/advisory

Generate AI farming advisory from city + crop
Main Endpoint. Send farmer's city, crop, state, and language → receive a contextual advisory text powered by Gemini Flash, live weather data, top news, and a generated audio file.
Request Body (JSON)
FieldTypeRequiredDescription
citystringRequiredFarmer's city (e.g. Ongole)
cropstringRequiredCrop name (e.g. rice)
statestringOptionalState name (e.g. Andhra Pradesh)
languagestringOptionalLanguage code: en/te/hi/... (default: en)
Clickable Examples (opens Swagger with pre-filled body)

{ "city": "Ongole", "crop": "rice", "state": "Andhra Pradesh", "language": "te" }

POST

/gemini/full-flow

Complete pipeline: audio in → AI advisory audio out
One-stop endpoint. Upload farmer audio → auto STT → extract city/crop → fetch live weather + news → Gemini advisory → TTS spoken audio.
Request — multipart/form-data + Query Params
ParamTypeRequiredDescription
audiofile (form)RequiredFarmer's audio clip
languagestring (query)OptionalLanguage code (default en)
citystring (query)OptionalOverride — skip STT extraction
cropstring (query)OptionalOverride — skip STT extraction
statestring (query)OptionalFarmer's state
